# 広島県 農業技術文書

## タスク

`hirosima/` 以下にあるPDFファイルを読み込んで、記載されている経営類型及び各類型で栽培する品目に関する経営指標に関する情報を抽出してください。

## 入力

- ファイルフォーマット: PDF（画像）
- 1つのファイルには、複数の経営類型が記載されています。経営類型ごとに栽培品目に関する経営指標が記載されており、各類型で栽培品目は1つ以上あります。


## 出力

経営類型に関する以下の4つの情報を抽出します。

- 前提表
- 栽培規模
- 経営収支
- 資本装備および減価償却費用

経営指標に関する以下の3つの情報を抽出します。

- 作業時間
- 作業技術
- 経営収支

出力のフォーマットについては README_SUBTASK1_SUBMISSION を参照ください。
