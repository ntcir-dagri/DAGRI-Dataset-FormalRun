# DAGRI Dataset FormalRun

このリポジトリでは、DAGRI の各サブタスクで用いる Formal Run 用データを提供しています。タスクへの参加方法やタスクの詳細は <https://sites.google.com/view/dagri/home-ja> を参照ください。

## DAGRIとは

DAGRI（Data Analytics for aGRicultural Information）は農業分野特有の知識のデジタル化および知識伝書の促進を目指し、図や表などの複雑なデータ表現を含む農業分野の文書を解析することを目的としたタスクで、3つのサブタスクから構成されます。

- Subtask 1: Table IE
  - フォーマットが異なる各都道府県の農業経営指標に関する文書から経営類型・経営指標に関する情報を抽出し，統一フォーマットに構造化を行うタスク
- Subtask 2: Multi-Table Multi-Hop QA
  - 農業分野の文書中の複数の表データをもとに、農作業に関する質問に回答するタスク
- Subtask 3: FiT-QA (Figures and Tables Question Answering)
  図や表が複雑に配置された、分野特有の表現を含む文書の内容に関する質問に回答するタスク
